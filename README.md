# AutoPayPlan UI Onboarding Layer

This repository contains the complete UI Onboarding Layer for the AutoPayPlan Financial Wellness app. Built with React and Vite, it delivers a high-fidelity, mobile-first experience across 41 interactive screens.

## 📖 Project Documentation
For a deep dive into the project's goals, technical implementation, and design decisions, please refer to the:
👉 **[Case Study: AutoPayPlan UI Onboarding Layer](./CASE_STUDY.md)**

## 🚀 Ready to Boot?
To run this project locally:
1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev`.

## ☁️ Vercel Deployment Steps
1. Push this code to your GitHub.
2. Log in to [Vercel](https://vercel.com) and click **"Add New Project"**.
3. Import this repository.
4. Click **"Deploy"**.

*This project was built focusing on mobile-responsive design, optimized for viewing on iOS and Android standard dimensions.*
